AIC Reading method

For screening read : Title, Journal, keywords and Abstract

For detailed synthesis use AIC:

  • Abstract → Summary of the paper🔎

  • Introduction → Context of problem and research, gap, summary of what is done

  • Conclusions → Summary of results and importance of results given the context.

Mental model when reading an abstract

  1. Topic → the writer provides background factual information.

  2. Problem → The writer presents the gap/problem to treat

  3. Research Gap → the writer states the limits of current knowledge.

  4. The purpose : the writer presents the general aim and the specific aim of the study.

  5. Methodology → the writer summarises the methodology and provides details.

  6. Results → the writer indicates the achievement of the study.

  7. Perspectives → the writer presents the implications of the study.
